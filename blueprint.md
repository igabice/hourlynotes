# Hourly Note App Blueprint

## Overview

Hourly Note is a simple yet powerful note-taking application designed for capturing thoughts, ideas, and reminders as they happen. The app provides a clean and intuitive interface for creating, viewing, and managing notes, with each entry timestamped for easy reference.

## Features

*   **Note Creation:** Quickly add new notes through a simple dialog.
*   **Note Display:** View a chronological list of all your notes.
*   **Note Deletion:** Easily remove notes you no longer need.
*   **Local Storage:** Notes are stored locally on your device using the Isar database.
*   **Themeing:** The app supports both light and dark themes.

## Design

*   **Color Scheme:** The app uses a modern and visually appealing color scheme with a primary color of deep purple.
*   **Typography:** The app uses the Oswald and Roboto fonts for a clean and readable look.
*   **Layout:** The app uses a simple and intuitive layout with a list of notes on the main screen and a floating action button for adding new notes.
*   **Components:** The app uses Material Design components for a consistent and familiar user experience.
